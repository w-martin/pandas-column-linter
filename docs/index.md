# typedframes

Static analysis for pandas and polars DataFrames. Catch column errors at lint-time, not runtime.

See the [README](https://github.com/will-martin/typedframes#readme) for a full introduction and
quick-start guide.

## Installation

```bash
pip install typedframes[pandas]   # pandas support
pip install typedframes[polars]   # polars support
pip install typedframes[mypy]     # mypy plugin
```
